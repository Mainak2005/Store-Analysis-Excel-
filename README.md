🛒 Store Sales Analysis & Excel Dashboard

📌 Project Overview
This project focuses on analyzing store sales data using Microsoft Excel and presenting insights through a dynamic and interactive dashboard. The aim is to transform raw data into meaningful business insights for better decision-making.

📊 Dataset
The dataset includes:
Order Details (Order ID, Order Date, Ship Date)
Customer Information (Customer Name, Segment, Region)
Product Details (Category, Sub-Category, Product Name)
Sales Metrics (Sales, Quantity, Profit, Discount)

🎯 Objectives

Perform data cleaning and preprocessing in Excel
Conduct Exploratory Data Analysis (EDA)
Identify top-performing products and customers
Analyze sales trends over time
Build an interactive Excel dashboard

🛠️ Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Cleaning Functions

📈 Key Insights
Top customers contributing the highest revenue identified
Most profitable product categories and sub-categories analyzed
Sales trends observed across different time periods
Regional performance comparison done
Impact of discounts on profit evaluated

📊 Dashboard Features
KPI cards (Total Sales, Profit, Quantity)
Region-wise sales analysis
Category & sub-category breakdown
Time-based trend analysis
Interactive slicers for filtering data
